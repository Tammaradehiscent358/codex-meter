## 📋 System requirements

This application functions on modern versions of macOS. You need an active internet connection to retrieve your usage data from the API. The app uses minimal system resources and stays out of your way while you work.

## 🔐 Privacy matters

We built this tool with privacy as the main goal. The app does not transmit your personal API keys or usage history to any server. Your data remains on your local disk at all times. You possess full control over the information the app reads and displays.

## 💡 Using the app

Once you launch the app, you see a small icon in the top right of your screen. Click this icon to open the main window. You need to enter your API key when you use the app for the first time. The app uses this key to talk to the service provider. 

If your usage hits a limit you set, the menu bar icon changes color to alert you. You can adjust these limits through the settings menu. Open the settings by clicking the gear icon inside the main window.

## ❓ Frequently asked questions

**Does this app slow down my computer?**
No. It consumes a very small amount of memory and processor power. 

**Can I track multiple API keys?**
Yes. You can add several accounts to the settings list and toggle between them.

**What happens if I forget my password?**
The app does not store passwords or accounts. It only uses API keys. If you lose your API key, generate a new one through your provider account dashboard.

**How do I delete the app?**
Move the app from your Applications folder into the Trash. Your saved configuration files live in your library folder. You can remove those separately if you wish to wipe all data.

**Does the app run in the background?**
The app stays in your menu bar. You can set it to start automatically when you turn on your computer. Find this option under the General tab in the settings window.

## 📦 Handling updates

The app checks for new versions when you open it. If a new version exists, the app notifies you. Click the button in the notification to download the installer for the new version. The installer replaces your old version automatically.

## 🔧 Troubleshooting

If the menu bar icon shows a warning sign, the app has trouble connecting to the service. Check your internet connection first. Ensure your API key has the necessary permissions. If the problem continues, quit the app and launch it again. Most connection errors resolve after a few seconds of network activity.
